Setup Instructions
1.Environment Configuration:
• Open the .env file and update the following variables:
• MONGODB_URL: Paste your respective MongoDB URL.
• STRIPE_SECRET_KEY: Paste your Stripe secret key.
• FRONTEND_URL: Specify the frontend URL where the frontend server will run.
2.Run Admin Backend:
• Navigate to the admin directory.
• Run the following command to start the admin backend server:
#npm run dev
3.Run Frontend:
• Navigate to the frontend directory.
• Run the following command to start the frontend server:
#npm run dev
4.Run Backend Server:
• Navigate to the backend directory.
• Run the following command to start the backend server:
#npm run server
5.Accessing the Application:
• The server will run on port 4000.
• Open the frontend URL provided after running the frontend server command in your web browser.
